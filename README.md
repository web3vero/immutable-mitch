# Immutable Mitch

A decentralized Web3 Chrome Extension tracking the status of Mitch McConnell, powered by a decentralized Web3 Oracle (Arweave/IPFS).

## 🚀 Quick Automated Install

You can easily download and extract Immutable Mitch by running the following command in your terminal:

```bash
curl -sSL https://raw.githubusercontent.com/web3vero/immutable-mitch/main/install.sh | bash
```

*After running the script, follow the on-screen instructions to load the unpacked extension into Chrome/Brave.*

## Introduction

In an era of misinformation and shifting political landscapes, knowing the exact status of key figures is crucial. **Immutable Mitch** leverages the power of decentralization—utilizing a robust Web3 Oracle built on Arweave and IPFS—to deliver verifiable, tamper-proof tracking of Mitch McConnell. By pulling data from immutable ledgers, this extension ensures the information you receive is permanent, transparent, and resistant to censorship.

## Manual Installation

To install **Immutable Mitch** as an unpacked extension in Google Chrome or Brave without the automated script, follow these step-by-step instructions:

1. **Download the Source Code:**
   Clone this repository or download the ZIP file and extract it to a known location on your computer.

2. **Open Extensions Page:**
   - In **Chrome**: Navigate to `chrome://extensions/` in your address bar.
   - In **Brave**: Navigate to `brave://extensions/` in your address bar.

3. **Enable Developer Mode:**
   Toggle the **"Developer mode"** switch located in the top-right corner of the Extensions page.

4. **Load the Extension:**
   Click the **"Load unpacked"** button that appears in the top-left area.

5. **Select the Directory:**
   Browse to the directory where you saved the **Immutable Mitch** source code, select the folder containing the `manifest.json` file, and click "Open" or "Select Folder".

6. **Pin the Extension:**
   For easy access, click the puzzle piece icon next to your address bar and click the pin icon next to **Immutable Mitch**.
